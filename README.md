## How to Add Your Profile Picture

1. Open the `images` folder.
2. Add your profile photo inside that folder.
3. Rename your photo to:

```text
profile.jpg
```

4. Keep the file path in `index.html` like this:

```html
<img src="images/profile.jpg" alt="Profile picture of Lee Jeong Shin A. Tero" class="profile-picture" />
```

If your image is PNG, either rename/convert it to `profile.jpg`, or change the code to:

```html
<img src="images/profile.png" alt="Profile picture of Lee Jeong Shin A. Tero" class="profile-picture" />
```
